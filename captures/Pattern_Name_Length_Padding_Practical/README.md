# Pattern Name Length / Padding / Practical Capture Set

This folder stores the 28 capture files for experiment C.

## Canonical naming

Use numeric canonical file names only:

- 001_CASE_001.syx
- 002_CASE_002.syx
- ...
- 028_CASE_028.syx

Do not use literal pattern-name strings as file names.

## Mapping source of truth

The case-to-name mapping and exact 16-byte intent are maintained in:

- datasets/analysis/pattern_name_length_padding_practical_20260526/case_mapping.yaml

## Capture order

Edit the same pattern repeatedly and export in order from case 001 to 028.
Re-import is not required between cases.
